WPShapere is a WordPress plugin to customize the WordPress Admin theme and elements as your wish.

== Description ==

WPShapere WordPress Admin Theme is a WordPress plugin and a powerful tool to customize your WordPress admin. It will totally white label the WordPress admin section. With WPShapere you have the ability to offer your customers a complete new admin dashboard with your brand name.
WPShapere has Simple to use user interface to manage the colors and some of the elements of the WordPress admin theme.

= Key Features =

* Unlimited color options + 5 pre-made Pro themes.
* Flat/Default design.
* White Label Branding.
* Upload custom logo for login and dashboard pages.
* RTL Compatibility.
* Custom login Theme.
* Hide unwanted widgets from dashboard.
* Manage Admin Bar elements.
* Hide WordPress version from the footer and add your company name or any text you want.
* Export and Import of settings feature.
* Powerful and simple to use Options panel.
* Add custom css styles to login and admin pages.
* Tested for Compatibility with popular plugins: Contact form 7, WP Super cache, WP Total cache, woocommerce, Elementor etc.

== Installation ==

1. Go to the 'Plugins' menu in WordPress and click 'Add New'
2. Search for 'WPShapere Lite' and select 'Install Now'
3. Activate the plugin by clicking the 'Activate' button.

== Frequently Asked Questions ==

= Can I set my own custom Logo in the WordPress admin bar? =
Yes, you can upload or link to external url of your own custom logo.

= Can I change the background color of WordPress admin bar? =
Yes, you can!

= Can I change the background color of WordPress admin menu sidebar? =
Yes, you can! You can also change menus background colors, menu text colors, etc.

= Is there Admin theme presets? =
Yes, you can switch between 5 preset themes.

= Can I customize the WordPress login page? =
Yes, you can customize the login background colors, text and menu colors.

= Can I set my own custom Logo in the WordPress login page? =
Yes, you can upload or link to external url of your own custom logo.

= Can I hide unwanted menu items from the admin bar?
Yes, you can!
